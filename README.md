# Personal Portfolio Website

A responsive personal portfolio website built with HTML5 and CSS3, featuring a clean dark theme design. This project showcases frontend development skills and serves as a professional online presence.

## Features

- **Responsive Design**: Optimized for various screen sizes
- **Sidebar Navigation**: Profile, contact information, and social links
- **Content Sections**:
  - About Me: Personal introduction and background
  - Skills: Technical skills with icons
  - Education: Academic background and achievements
  - Portfolio: Featured projects with descriptions

## Technologies Used

- HTML5
- CSS3 (Grid and Flexbox layouts)
- Font Awesome icons (via CDN links)

## Installation & Usage

1. Clone or download the repository
2. Open `project.html` in any modern web browser
3. No additional setup required - it's a static website

## Project Structure

```
project.html    # Main HTML file
project.css     # Stylesheet
README.md       # This file
```

## Browser Support

Compatible with all modern browsers including Chrome, Firefox, Safari, and Edge.
